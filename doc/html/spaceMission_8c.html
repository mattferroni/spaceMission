void checkCollision ( )

Check for a collision with an asteroid or a cherry The spacecraft collides with something if:

  • it's in position 1 and there's an object in line 1;
  • it's in position 0 and there's an object in line 0. If the collision is with a cherry, points are incremented by CHERRYPOINTS

void interrupt isr ( void  )

Interrupts handling routine.

This routine is used to handle timing logic. updateDivider is incremented at every iteration until it reaches the refDivider: the space environment is then shifted by one position. divider is incremented at every iteration until it reaches the DIVIDER: the countdown time is then decremented by one

References c3, c4, countdown, DIVIDER, divider, ENDGAME, refDivider, status, UPDATE, and updateDivider.

int main ( )

Main of the program. Initialize registers, clear the LCD display and initialize the countDown stuff. The main loop:

  1. displays asteroids and shows them on LCD display, if something changed;
  2. checks if countdown reached 0 or the spacecraft collided with an asteroid;
  3. checks if a button up has been pressed and pushes up the spacecraft;
  4. checks if a button down has been pressed and pushes down the spacecraft;
  5. polls the ADC to set asteroids speed;
  6. displays side digits (points and countDown).

void updateSpace ( )

Update the asteroids and the score points.

This function use 2 "circular" buffers (one for each line) of 16 digits each (line1 and line0)

  1. Increments the points when an asteroids reach the end of the line.
  2. Generate a new asteroid randomly on line0[indexLines] if no asteroid have been generated on line1[prevIndex] and vice-versa to avoid impossible paths for the spacecraft.
  3. Generates randomly a cherry instead of an asteroids
  4. Check for collisions
